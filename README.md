# Customizable react carousel

Running examples locally:

  - clone project
  - run ```yarn```
  - run ```yarn dev-server```
  - open ```localhost:8080```

### Installation
  - Install package
        ```
        yarn add react-customizable-carousel
        ```
  - Import component
    ```
    import { Carousel } from 'react-customizable-carousel'
    ...

    <Carousel>
      <h1>1</h1>
      <h1>2</h1>
      <h1>3</h1>
      <h1>4</h1>
      <h1>5</h1>
      <h1>6</h1>
    </Carousel>
    ```
  - Import CSS:
        ```
        @import "react-customizable-carousel";
        ```
